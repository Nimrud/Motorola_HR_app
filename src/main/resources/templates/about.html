Prosta aplikacja do zarządzania pracownikami w firmie. Pozwala wyświetlić ich listę, dodać nowych pracowników, wyświetlić listę przelewów itp.

  • Java 11+
  • Spring (Web, Data JPA, Boot, Security)
  • H2
  • Hibernate Validator
  • Lombok
  • Thymeleaf
  • HTML 5
  • CSS
